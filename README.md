# Инструкция по запуску
## Версия Java
Java SE 20
## Система сборки
В качестве системы сборки используется Apache Maven 3.9.6

## Зависимости 
- JUnit 5 (JUnit Jupiter)
- Apache Commons CLI

Более подробно в [pom.xml](https://github.com/dmc3105/cft-shift-test-task/blob/master/pom.xml)

## Сборка проекта
```
mvn clean package
```
